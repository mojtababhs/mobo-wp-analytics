# MoBo WP Analytics 2.0.0

افزونه حرفه‌ای آمار بازدید وردپرس با داشبورد مدرن و امکانات پیشرفته.

## شورت‌کد جدید
```text
[mobo_wp_analytics range="7days"]
```

## امکانات اصلی
- بازدیدها، بازدیدکننده یکتا، نشست‌ها و آنلاین‌ها
- تفکیک بازدید مدیر، مهمان و کاربران عضو
- تنظیم محاسبه یا عدم محاسبه هر نوع کاربر
- ورودی موتورهای جستجو، شبکه‌های اجتماعی، رفرال، مستقیم و کمپین
- پشتیبانی از UTM: source, medium, campaign, term, content
- شناسایی خزنده‌های وب
- سیستم‌عامل، مرورگر، دستگاه و کشور
- پربازدیدترین صفحات، صفحات ورود و صفحات خروج
- نرخ پرش تقریبی بر اساس نشست‌های تک‌صفحه‌ای
- گزارش جستجوی داخلی وردپرس
- گزارش خطاهای 404
- نمودارهای داشبورد با Chart.js
- خروجی CSV سازگار با Excel
- خروجی JSON
- REST API مدیریتی: /wp-json/mobo-wp-analytics/v1/summary
- پاکسازی خودکار داده‌های قدیمی
- تنظیمات حریم خصوصی و عدم ذخیره User-Agent کامل

## نکته
برای تشخیص کشور، افزونه از هدرهای CDN مثل Cloudflare استفاده می‌کند. برای نسخه دقیق‌تر می‌توان MaxMind GeoIP را اضافه کرد.
