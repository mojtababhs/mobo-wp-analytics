## 📦 Installation

### Method 1 — Install from WordPress Dashboard

1. Go to WordPress Admin Dashboard.
2. Navigate to Plugins → Add New.
3. Upload the plugin ZIP file.
4. Click Install.
5. Activate the plugin.

---

### Method 2 — Manual Installation

1. Extract the ZIP archive.
2. Upload the `mobo-wp-analytics` folder into:

```text
/wp-content/plugins/
```

3. Open WordPress Admin Dashboard.
4. Activate `MoBo WP Analytics` from Plugins section.

---

## 🚀 Usage

After activation:

1. A new menu called `MoBo Analytics` will appear.
2. Open the analytics dashboard.
3. Select your desired date range.
4. Start analyzing your website traffic.

---

## 🧩 Shortcode

```text
[mobo_wp_analytics range="7days"]
```

Supported ranges:

```text
today
yesterday
7days
30days
90days
all
```

---

## 🔌 REST API

```text
/wp-json/mobo-wp-analytics/v1/summary
```

Perfect for dashboards, mobile apps and external monitoring systems.

---

## 🔒 Privacy

- Raw IP addresses are not stored.
- Visitor identifiers are hashed.
- User-Agent storage can be disabled.
- Admin or logged-in user tracking can be disabled.

# 🚀 نصب افزونه

## روش اول: نصب از طریق وردپرس

1. وارد پیشخوان وردپرس شوید.
2. از منوی «افزونه‌ها» روی «افزودن» کلیک کنید.
3. فایل ZIP افزونه را بارگذاری کنید.
4. روی «نصب افزونه» کلیک کنید.
5. پس از پایان نصب، افزونه را فعال کنید.

---

## روش دوم: نصب دستی

1. فایل ZIP افزونه را استخراج کنید.
2. پوشه `mobo-wp-analytics` را در مسیر زیر قرار دهید:

```text
/wp-content/plugins/
```

3. وارد پنل مدیریت وردپرس شوید.
4. از بخش افزونه‌ها، افزونه `MoBo WP Analytics` را فعال کنید.

---

# ⚙️ فعال‌سازی افزونه

پس از فعال‌سازی:

1. منوی جدیدی با نام `MoBo Analytics` به پنل وردپرس اضافه می‌شود.
2. وارد داشبورد افزونه شوید.
3. بازه زمانی موردنظر را انتخاب کنید.
4. آمار بازدید و نمودارها به‌صورت زنده نمایش داده می‌شوند.

# 🔒 حریم خصوصی

- IP خام کاربران ذخیره نمی‌شود.
- شناسه بازدیدکننده به‌صورت هش‌شده ذخیره می‌شود.
- امکان غیرفعال‌کردن ذخیره User-Agent وجود دارد.
- امکان غیرفعال‌کردن رهگیری مدیران یا کاربران عضو وجود دارد.
